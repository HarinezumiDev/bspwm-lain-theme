## Serial Experiments Lain Configuration

### Packages
- bspwm
- sxhkd
- eww
- fastfetch
- picom
- polybar
- rofi
### Platforms
- desktop
- laptop
_Choose this by using `install.sh`_
### Setup
1. Clone the repository to your local machine
```bash
git clone https://github.com/HarinezumiDev/bspwm-lain-theme
```
2. Enter the directory
```bash
cd bspwm-lain-theme
```
3. Give execution privelegies to installation script
```bash
chmod +x install.sh
```
4. Run the script
```bash
./install.sh
```
-> 4.1 The script will warn you about changing files in `.config`. Make the backups of your configurations first. To contionue type _y_ and press Enter
-> 4.2 Choose your platform - desktop/laptop. This changes whether to add batery display to polybar

READY!
